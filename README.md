# antfarm

## Основные факторы:
Рендеринг шаблонов html
SEO оптимизация
Админ панель
Интеграция в будущем с доставщиком
Запросы на почту о заказе


## Принцип работы
Пользователь выбирает корзину и переходит к покупке -> заполняет информацию о себе (почта, номер) -> это отправялется на бекенд -> сервер обрабатывает и создает заказ у доставщика, записывает в базу данных информацию -> пользователю на почту отсылается трек номер заказа от доставщика и контактная информация и информация о его заказе

## Библиотеки
SQL-first для создания библиотеки
```
go install github.com/sqlc-dev/sqlc/cmd/sqlc@latest
```
GIN для бекенд сервера
```
go install github.com/gin-gonic/gin@latest
```
Загрузка .env
```
go install github.com/joho/godotenv/cmd/godotenv@latest
```
